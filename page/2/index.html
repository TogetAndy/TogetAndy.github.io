《写给自己的当某天》

呱呱坠地
我们只是一张白纸
童年的每一笔，黯淡在岁月的风中
默然回首
我们会有怎样词穷

当某天
看见某个小屁孩
趴在桌子上，留下一大滩口水

当某天
天空飘起大雪
穿着雨鞋的他，脸蛋冻得通红却笑开了花

当某天
马路上，路灯下
骑自行车的影子
与你擦肩而过

当某天
空旷的走廊上
天真烂漫的他和她
一起趴在栏杆上

当某天
喧闹的沙滩旁
满怀报复的他和他
大声喊道：”沙滩一趟三年半
，大浪来时我翻身”

当某天
我踏上公交
不为兼程，只为寻一个你

当某天
回到这到这操场
希望，那一刻的决定，不会让我后悔

当某天
承诺拒绝喝酒
却被抗回这3栋628
叫嚣着：“捞比（室友爱称），我要喝水”

当某天
凌晨两点，酒醒过后
发现自己四年的大学生涯真的结束了

当某天，
你看到当某天。

当某天......
翻开多年以后的它。
希望它是精彩的诗篇。

Config_Hexo

软件需求

  • git
  • Node.js

hexo基本安装配置

  • 安装cnpm(淘宝镜像):

    • npm install -g cnpm –registry=https://registry.npm.taobao.org

  • 安装hexo:

    • cnpm install -g hexo-cli
    • cnpm install hexo -save
    • hexo -v

  • 本地初始化：

    • hexo init

  • 生成安装器： cnpm install

测试

本地测试：
    hexo clean
    hexo g
    hexo s

发布博客相关设置：
    安装hexo git插件
    npm install hexo-deployer-git --save

    设置用户名，密码：
    git config --global user.name "你的用户名"
    git config --global user.email "你的邮箱"

发布：
    hexo clean
    hexo g
    hexo d
